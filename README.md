# ArduinoLCD-VoltageRead-Buzzer
## About - 説明
ArduinoでLCDの表示、電圧測定、ブザーで音を鳴らすといった動作をします。  
The Arduino will be used to display the LCD, measure the voltage, and sound the buzzer.
